Welcome to Kitchen Data.

If you've ever tried writing a cooking app, whether it's for mobile or for web, then you've inevitably found yourself trying to gather information across the web about basics like measurement conversions, substitutions, generic cooking times, etc.

This repo is meant to provide a hub for some of this common data, but in a semantic-friendly, reusable format. Currently, we plan to support three formats -- JSON, PLIST and XML -- emphasizing web and mobile-first.

We're just getting started with this, so please send me a message (@wubbahed) if you have any questions or would like to contribute. All data sets will be MIT Licensed and freely available to use.
